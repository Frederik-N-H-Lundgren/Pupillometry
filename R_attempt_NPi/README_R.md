# Exploratory NPi Reconstruction (R)

This folder contains the initial exploratory work on reconstructing the proprietary Neurological Pupillary Index (NPi) using **R**.

The analyses primarily focused on generalized additive models (GAMs) and related statistical approaches to investigate the relationship between pupillometric features and the NPi score. These notebooks and scripts were part of the early development phase of the project and were used to explore potential modelling strategies and feature relationships.

The work in this folder is **exploratory only** and **was not used for the final results presented in the thesis**. The final NPi reconstruction was implemented using the Python framework contained in the main repository.

The original clinical data are not included due to patient privacy regulations, and some outputs have therefore been removed or limited.
